# Chirp 系列 TTS 使用指南

## 快速开始

### 1. 管理界面配置渠道

| 字段 | 值 |
|------|-----|
| 类型 | Google Cloud TTS (Chirp) |
| 名称 | 自定义（如 "Google TTS"） |
| 密钥 | Google Cloud API Key（需启用 Cloud Text-to-Speech API） |
| 代理 | 留空（默认 `https://texttospeech.googleapis.com`） |
| 模型 | chirp-3, chirp-2, chirp_telephony |

### 2. 获取 API Key

```bash
# 启用 Cloud Text-to-Speech API
gcloud services enable texttospeech.googleapis.com

# 创建 API Key（限定仅 TTS 服务）
gcloud services api-keys create \
  --display-name="oneapi-chirp-tts" \
  --api-target=service=texttospeech.googleapis.com
```

### 3. 调用接口

```bash
curl -X POST https://oneapi.gitagent.io/v1/audio/speech \
  -H "Authorization: Bearer sk-YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "chirp-3",
    "input": "Hello, this is a test.",
    "voice": "alloy"
  }' \
  --output output.mp3
```

## 支持的模型

| 模型 | 说明 | 计费 |
|------|------|------|
| `chirp-3` | Chirp 3 HD，最新高质量语音，23+ 个语音 | $30/1M 字符 |
| `chirp-2` | Chirp 2 HD，上一代，3 个语音（D/F/O） | $16/1M 字符 |
| `chirp_telephony` | 电话场景优化，低延迟，3 个语音 | $16/1M 字符 |

---

## 请求参数

完全兼容 OpenAI `/v1/audio/speech` 接口：

```json
{
  "model": "chirp-3",
  "input": "要合成的文本",
  "voice": "alloy",
  "speed": 1.0,
  "response_format": "mp3"
}
```

| 参数 | 必填 | 说明 |
|------|------|------|
| `model` | 是 | chirp-3 / chirp-2 / chirp_telephony |
| `input` | 是 | 要合成的文本（最大 4096 字符） |
| `voice` | 是 | 语音名称，支持三种格式（见下方） |
| `speed` | 否 | 语速，范围 0.25-4.0，默认 1.0 |
| `response_format` | 否 | 输出格式，默认 mp3 |

### 输出格式

| response_format | 实际编码 | Content-Type |
|-----------------|---------|-------------|
| `mp3`（默认） | MP3 | audio/mpeg |
| `opus` | OGG_OPUS | audio/ogg |
| `wav` | LINEAR16 | audio/wav |
| `pcm` | LINEAR16 | audio/wav |
| `aac` | MP3（降级） | audio/mpeg |
| `flac` | LINEAR16（降级） | audio/wav |

## 语音选择

### 方式一：OpenAI 兼容语音名

直接使用 OpenAI 的语音名，自动映射为对应的 Chirp 语音：

**Chirp 3 映射：**

| OpenAI 语音 | Chirp 3 语音 | 风格 |
|-------------|-------------|------|
| `alloy` | Kore | 中性平衡 |
| `echo` | Charon | 低沉男声 |
| `fable` | Achernar | 叙述风格 |
| `onyx` | Fenrir | 深沉有力 |
| `nova` | Leda | 明亮女声 |
| `shimmer` | Aoede | 柔和温暖 |

**Chirp 2 映射：**

| OpenAI 语音 | Chirp 2 语音 | 性别 |
|-------------|-------------|------|
| `alloy` / `nova` | F | 女声 |
| `echo` / `onyx` | D | 男声 |
| `fable` / `shimmer` | O | 女声 |

示例：
```json
{"model": "chirp-3", "voice": "alloy", "input": "Hello"}
```

### 方式二：Chirp 原生语音名

直接传入 Chirp 语音名（不含语言前缀），自动拼接为完整名称：

```json
{"model": "chirp-3", "voice": "Charon", "input": "Hello"}
```
等价于 `en-US-Chirp3-HD-Charon`。

### 方式三：完整语音名（支持多语言）

传入含语言代码的完整语音名，系统自动提取语言代码：

```json
{"model": "chirp-3", "voice": "cmn-CN-Chirp3-HD-Kore", "input": "你好"}
```
```json
{"model": "chirp-3", "voice": "ja-JP-Chirp3-HD-Aoede", "input": "こんにちは"}
```

## 代码示例

### Python

```python
import requests

response = requests.post(
    "https://oneapi.gitagent.io/v1/audio/speech",
    headers={
        "Authorization": "Bearer sk-YOUR_TOKEN",
        "Content-Type": "application/json",
    },
    json={
        "model": "chirp-3",
        "input": "你好，这是 Chirp 3 语音合成测试。",
        "voice": "cmn-CN-Chirp3-HD-Kore",
        "response_format": "mp3",
    },
)

with open("output.mp3", "wb") as f:
    f.write(response.content)
```

### Node.js (OpenAI SDK)

```javascript
import OpenAI from "openai";
import fs from "fs";

const openai = new OpenAI({
  apiKey: "sk-YOUR_TOKEN",
  baseURL: "https://oneapi.gitagent.io/v1",
});

const response = await openai.audio.speech.create({
  model: "chirp-3",
  voice: "alloy",
  input: "Hello, this is a Chirp 3 test.",
});

const buffer = Buffer.from(await response.arrayBuffer());
fs.writeFileSync("output.mp3", buffer);
```

## 故障排查

| 错误信息 | 原因 | 解决方案 |
|---------|------|---------|
| `Voice 'xxx' does not exist` | 语音名拼写错误或该模型不支持 | 检查语音名格式，Chirp 2 只有 D/F/O |
| `language code doesn't match` | 语音名中的语言与 languageCode 不一致 | 使用完整语音名（含语言前缀） |
| `该令牌无权使用模型` | Token 的模型白名单未包含 Chirp 模型 | 在管理界面编辑 Token，添加 chirp-3 等模型 |
| `API key not valid` | API Key 无效或未启用 TTS 服务 | 检查 Google Cloud Console 中 API 状态 |
| `PERMISSION_DENIED` | API Key 无权限调用 TTS | 确认 API Key 已关联 Text-to-Speech API |
